Installation

CocoaPods

You want to add pod "FlyveMDMInventory", "~> 1.2" similar to the following to your Podfile:
{% highlight swift %} target "MyApp" do pod "FlyveMDMInventory", "~> 1.2" end {% endhighlight %}
Then run a pod install inside your terminal, or from CocoaPods.app.
Alternatively to give it a test run, run the command:
{% highlight swift %} pod try FlyveMDMInventory {% endhighlight %}
Page on CocoaPods.org Documentation

Carthage

Install using Carthage by adding the following lines to your Cartfile:
{% highlight swift %} github "flyve-mdm/flyve-mdm-ios-inventory" "master" {% endhighlight %}
Then, run the following command:
{% highlight swift %} carthage update --platform iOS {% endhighlight %}
  • On your application targets “General” settings tab, in the “Linked Frameworks and Libraries” section, drag and drop each framework you want to use from the Carthage/Build folder on disk.
  • On your application targets “Build Phases” settings tab, click the “+” icon and choose “New Run Script Phase”. Create a Run Script with the following contents: {% highlight swift %} /usr/local/bin/carthage copy-frameworks {% endhighlight %}
  • And add the paths to the frameworks you want to use under “Input Files”, e.g.: {% highlight swift %} $(SRCROOT)/Carthage/Build/iOS/FlyveMDMInventory.framework {% endhighlight %}
